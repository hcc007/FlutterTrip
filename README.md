# flutter_trip

[慕课网仿携程网App](https://coding.imooc.com/class/321.html)，基于Flutter支持Android与iOS平台。

#### 一、运行
1、正确配置 Flutter 开发环境

2、国内需要添加 Flutter 中国镜像环境变量

3、android 环境最好配 阿里云 仓库

4、flutter pug get 下载第三方依赖（Flutter 1.17 能够正确运行，Flutter sdk 版本低可能需要升级版本）
